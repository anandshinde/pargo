### LINK COMPOUND

Link Compound is a dumb UI component that handles displaying link compounds
from Bloomreach.

**Extends:**

- Simple Component base class

**Inputs:**

- content:
  ```javascript
  {
   label: 'Button Label',
   icon?: 'search';
  }
  ```
- presentation :
  ```javascript
  {
      theme: 'dark',
      buttonType: 'primary'
  }
  ```
